## 概要

新しい人・視点・スキルを組織に取り込むことで変革を起こす手段。
採用形態によってスピード・コスト・リスクが大きく異なる。

## 登用の種類

| 種類 | 特徴 |
|------|------|
| 新卒採用 | 文化醸成・長期育成向き |
| 中途採用 | 即戦力・専門スキル向き |
| アルバイト | 短期・特定業務向き |
| インターン | 低コスト・相互評価向き |
| 副業採用 | 専門性×低コミット・変革触媒向き |
