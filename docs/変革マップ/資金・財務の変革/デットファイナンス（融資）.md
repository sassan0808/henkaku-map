## 概要

金融機関から借入を行い、返済義務と引き換えに事業資金を調達する手段。
株式希薄化なく資金を得られるのが特徴。

## 主な調達先

| 種別 | 特徴 |
|------|------|
| 日本政策金融公庫 | 創業期・中小企業向け。低金利・無担保で借りやすい |
| 信用保証協会付き融資 | 民間銀行でも保証付きで借りやすくなる |
| 民間銀行（プロパー融資） | 実績ある企業向け。審査は厳しいが条件が良い |
| ビジネスローン | スピード重視。金利は高め |
| ノンバンク系 | 審査が柔軟。短期資金調達に向く |

## 活用のポイント

- **財務諸表の整備**：決算書・試算表の質が審査に直結する
- **資金使途の明確化**：何に使うかを説明できることが重要
- **借入タイミング**：必要になってから動くと遅い。余裕があるときに動く
- **返済計画**：キャッシュフロー計画と合わせて設計する

## こんな課題に効く

- 設備投資・在庫仕入れに資金が必要
- 売上はあるが入金タイミングがズレてキャッシュが不足
- 創業・新規事業の立ち上げ資金が必要
