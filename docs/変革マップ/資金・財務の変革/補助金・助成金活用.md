## 概要

国・都道府県・市区町村が提供する公的資金支援を活用し、設備投資・DX推進・人材育成などのコストを補填する手段。
返済不要な点が最大のメリット。

## 代表的な補助金・助成金

| 名称 | 対象 | 補助上限 |
|------|------|---------|
| ものづくり補助金 | 設備投資・システム開発 | 最大1,250万円 |
| IT導入補助金 | ITツール・SaaS導入 | 最大450万円 |
| 事業再構築補助金 | 新分野展開・業態転換 | 最大1.5億円 |
| キャリアアップ助成金 | 非正規→正規転換・処遇改善 | 最大80万円/人 |
| 雇用調整助成金 | 雇用維持 | 最大15,000円/日 |
| 小規模事業者持続化補助金 | 販路開拓・設備投資 | 最大250万円 |

## 活用のポイント

- **申請タイミング**：公募期間が決まっており、逃すと次回まで待つ必要がある
- **事前準備**：事業計画書の質が採択率に直結する
- **専門家活用**：中小企業診断士・行政書士に申請支援を依頼するケースが多い
- **補助金は先払い不可**：一度自己負担し、後から補填される仕組みが基本

## こんな課題に効く

- DXツールを導入したいが予算が足りない → IT導入補助金
- 新しい設備・システムに投資したい → ものづくり補助金
- 新規事業に踏み出したい → 事業再構築補助金
- 採用・人材育成コストを抑えたい → 各種助成金
