## 概要

株式と引き換えに投資家から資金を調達する手段。
返済不要だが、株式の希薄化と投資家との関係管理が伴う。

## 主な調達先

| 種別 | 特徴 |
|------|------|
| ベンチャーキャピタル（VC） | 成長性重視。Exit（IPO・M&A）を前提に投資 |
| エンジェル投資家 | 個人投資家。少額から。メンタリングも期待できる |
| CVC（コーポレートVC） | 事業会社系。資金＋事業連携が期待できる |
| 事業会社からの出資 | アライアンスと組み合わさることが多い |
| クラウドエクイティ | 小口の一般投資家から調達 |

## 活用のポイント

- **投資家選び**：お金だけでなく、知見・ネットワーク・バリューアップ支援を見る
- **バリュエーション**：会社の評価額交渉が資本政策に長期的に影響する
- **資本政策の設計**：誰にどの順番でどれだけ渡すかを早めに設計する
- **VC向けピッチ**：市場規模・差別化・チームの3点が特に重視される

## こんな課題に効く

- 急成長を目指すため大きな資金が必要
- 資金だけでなく業界知見・人脈も欲しい
- 銀行融資では対応しにくいリスクの高い事業に投資したい
