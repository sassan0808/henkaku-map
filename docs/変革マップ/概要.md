## 変革手段の8つの柱

課題の構造に対応した変革手段を8つの柱で体系化しています。

| 柱 | 課題 | 代表的な手段 |
|---|---|---|
| [知見・意思決定の変革](/docs/変革マップ/知見・意思決定の変革/概要) | 何をすべきか判断できない | スポットコンサル、市場調査 |
| [人・能力の変革](/docs/変革マップ/人・能力の変革/概要) | 人の質・意識を変えたい | 社内研修、越境学習 |
| [体制・人材の変革](/docs/変革マップ/体制・人材の変革/概要) | 必要なリソースが社内にいない | 人材登用、副業・フリーランス、顧問 |
| [実行・プロセスの変革](/docs/変革マップ/実行・プロセスの変革/概要) | 実行が伴わない・効率が悪い | ツール導入、BPR、PMO |
| [売上・収益の変革](/docs/変革マップ/売上・収益の変革/概要) | 売上が伸びない・収益エンジンが機能していない | 営業プロセス改革、マーケティング変革、CS、RevOps |
| [販路・外部資本の変革](/docs/変革マップ/販路・外部資本の変革/概要) | 自社単独の成長に限界がある | 販路開拓、アライアンス、M&A |
| [資金・財務の変革](/docs/変革マップ/資金・財務の変革/概要) | 資金が足りない、財務の専門性がない | 補助金、融資、出資、クラファン |
| [データ・AI活用の変革](/docs/変革マップ/データ・AI活用の変革/概要) | AIを組織全体で使いこなせていない | データ基盤整備、AIリテラシー研修、AI活用推進 |

## 使い方

課題が明確な場合はここから手段を探せます。
「何に困っているか」が先の場合は [課題マップ](/docs/課題マップ/概要) からスタートしてください。
