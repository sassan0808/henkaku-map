## 概要

正社員として新しい人材を採用し、組織に新しい視点・スキルを注入する手段です。
即戦力（中途）と長期育成（新卒）でアプローチが異なります。

## 中途採用 vs 新卒採用の使い分け

| 観点 | 中途採用 | 新卒採用 |
|------|---------|---------|
| 即戦力性 | 高い（経験・スキルを持ち込む） | 低い（育成に時間がかかる） |
| 文化への影響 | 既存文化を揺さぶる可能性がある | 組織文化に染まりやすい |
| コスト | 採用単価が高い傾向 | 採用単価は低いが育成コストが高い |
| 向くケース | 専門スキル・経験が即必要 | 中長期の組織強化・文化醸成 |

変革スピードを求めるなら中途、組織の底上げを長期で狙うなら新卒です。多くの場合は両方を組み合わせます。

## 採用が失敗するパターン

- **ペルソナが曖昧** — 「優秀な人が欲しい」だけでは採れない。どんな課題を解ける人が必要かを具体化する
- **ミスマッチな期待値** — 会社側が「何でもできる人」を期待し、入社後にギャップが生じる
- **オンボーディングがない** — 採用して終わり。受け入れ設計がなく、早期離職につながる
- **採用ブランドを軽視している** — 候補者は採用プロセスで会社を評価している。対応・コミュニケーションが雑だと優秀な人が離れる

## 採用ブランドの重要性

「採れない」の多くは、採用チャネルより**採用ブランド（会社として人が行きたいと思えるか）**の問題です。

強い採用ブランドの要素：
- 明確なミッション・ビジョン
- 具体的な成長機会とキャリアパス
- 社員のリアルな声（SNS・採用媒体）
- 経営者・チームのカルチャーの透明性

## 「副業・フリーランス活用」との使い分け

正社員採用には時間・コストがかかります。まず副業・フリーランスで試してから正社員採用するアプローチも有効です。

```
副業で3〜6ヶ月関わる
  → 実力・カルチャーフィットを確認してから
  → 正社員として迎える（リスク低減）
```

## こんな課題に効く

- 特定スキル・経験が組織に完全に欠けている
- 外部の視点・文化を持ち込んで組織変革の起点にしたい
- 長期的な組織の底上げ・文化づくりをしたい
- 副業・フリーランスでは補えない「コアメンバー」が必要
