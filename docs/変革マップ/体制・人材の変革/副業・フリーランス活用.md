## 概要

即戦力のプロ人材を必要な期間・範囲でプロジェクトに組み込む手段。
正社員採用より低リスク・低コストで高い専門性を調達できる。

## 人・能力の変革との接点

副業人材の知見を社内に移転する設計をすることで、
実質的な研修・リスキリング効果が得られる。

- 副業人材をメンターとして位置づける
- 社内向けワークショップのファシリテーターに起用する
- ドキュメント整備を通じて暗黙知を形式知化する

## 実行・プロセスの変革との接点

新ツール・新プロセスに精通した副業人材を招き、導入を主導してもらう。

- ツール選定・導入設計のアドバイザーとして関わってもらう
- 社内向けマニュアル・研修コンテンツを作ってもらう

## サービスの選び方

```
何をしたいか？
├── ビジネス課題の壁打ち・戦略支援がほしい → Lotsful
├── エンジニア・PM・デザイナーを副業で → Offers / Workship
├── エンジニアをフルタイムに近い形で → レバテックフリーランス
└── 単発タスクを安く処理したい → クラウドワークス / Lancers
```
