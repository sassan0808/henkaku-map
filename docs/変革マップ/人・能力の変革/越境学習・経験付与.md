## 概要

社員を社外・異業界・異文化の環境に置くことで、マインドセットを強制的に変える手段です。
研修では変えられない「当たり前」の枠を壊すことを目的とします。

## なぜ研修ではなく「越境」が必要か

社内での研修・学習は、既存の前提を強化することに向いています。しかし変革に必要なのは、**「今の会社の常識が常識ではない」という体験**です。それは外の世界に出ることでしか得られません。

```
社内研修          → スキルを磨く。前提は変わらない
越境（外に出る）   → 「当たり前」が相対化される。マインドセットが変わる
```

## 越境の種類

| パターン | 内容 | 効果 |
|---------|------|------|
| **他社出向・兼務** | 別の組織で一定期間働く | 異なる組織文化・判断基準を体感する |
| **副業・社外プロジェクト** | 自社外でプロジェクトに関与する | 自分の市場価値・スキルの客観視 |
| **スタートアップ・新規事業への参加** | 変化の速い環境に身を置く | 自律・実行のマインドセットが育つ |
| **NPO・社会活動** | 異なる価値観の人々と働く | 多様性への理解・視野の拡大 |
| **異業種交流・コミュニティ** | 異業界の人との接点を継続的に持つ | 自社業界の「当たり前」を相対化する |
| **海外研修・グローバル体験** | 異文化環境に身を置く | 文化・制度の多様性を体感する |

## 「越境学習」として機能させるための設計

出すだけでは効果が出ません。事前・事後の設計が重要です。

1. **目的の明確化** — 何を学ばせたいか、どう変わってほしいかを具体的に設定する
2. **事前の期待設定** — 本人と、なぜこの機会を設けるかを共有する
3. **定期的な振り返り** — 越境中に感じた違和感・発見を言語化する場を設ける
4. **社内への還元設計** — 越境後に得た視点・経験を組織にフィードバックする場を用意する

## こんな効果が生まれる

- 変革への抵抗感が薄れる（「できない」という思い込みが崩れる）
- 危機感・主体性が生まれる（外に出て初めて「自社の危うさ」を実感する）
- 新しいネットワークが組織に入ってくる
- 「意欲ある人材」の流出防止になる（挑戦できる場があることで定着）

## こんな課題に効く

- 変革への抵抗が強く、「うちには無理」という空気がある
- 優秀な社員が「社内に成長の場がない」と感じて離職している
- 視野が社内・業界内に閉じていて、新しい発想が生まれない
- リーダー候補の育成が進まない
