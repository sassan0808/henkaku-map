## 概要

ターゲット企業へのアポイントメントや商談機会を外部の力で創出する手段。
自社営業リソースが限られる場合や、新規領域への展開に有効。

## 主な手段

| 手段 | 内容 | 向くケース |
|------|------|---------|
| **営業代行・アポ獲得支援** | テレアポ・メール営業を外部に委託 | 自社営業リソースが足りない |
| **マッチングプラットフォーム** | BtoBマッチングサービスで接点を作る | 特定業界・規模の企業との接点がない |
| **展示会・イベント出展** | ターゲット顧客が集まる場に出展する | 一度に多くの見込み客にリーチしたい |
| **業界団体・協会への参加** | 業界のネットワークを活用する | 信頼関係が先行する業界での開拓 |
| **紹介・エージェント活用** | 既存顧客や顧問の人脈を活用する | 信頼が重要な業種（不動産・金融・士業等） |

## 「外部頼み」にならないための原則

販路開拓支援は「接点を作る」ことはできるが、「売れる」かどうかは自社の実力次第。

- アポを取っても成約しないなら、**提案内容・ターゲティングの問題**
- 商談が増えても顧客に刺さらないなら、**価値提案の問題**
- 継続・紹介が起きないなら、**プロダクト・サービス品質の問題**

外部の力で接点を増やしながら、並行して自社の営業力・提案力を高めることが重要。

## こんな課題に効く

- 新規業界・地域への参入で、一からネットワークを構築する時間がない
- 営業人材がいないが、受注を増やさなければいけない
- 既存の営業チャネルが頭打ちになっている
