## 概要

物理的な拠点（店舗・営業所・ショールーム等）を新たに設けることで、
アクセスできる顧客・地域・市場を拡大する手段。
フランチャイズ展開や代理店網の構築なども含む。

## アプローチの種類

| アプローチ | 内容 | 向いているケース |
|-----------|------|---------------|
| 直営店の新規出店 | 自社運営の店舗を新たに開く | ブランド・品質を自社でコントロールしたい |
| フランチャイズ（FC）展開 | 加盟店に自社モデルを展開する | 自己資本・人員を抑えて急拡大したい |
| ポップアップストア | 期間限定の出店で市場検証 | 新エリア・新顧客層のテストとして |
| 代理店・販売店の開拓 | 既存業者に自社商品を取り扱ってもらう | BtoBの販路拡大・地方展開 |
| ショールーム・体験拠点 | 購買よりも体験・認知が目的の拠点 | 高関与商材・ブランディング重視 |

## 出店・展開を成功させるポイント

- **立地選定の精度** — 商圏・競合・顧客導線を徹底的に分析する
- **オペレーションの標準化** — 展開前に「1店舗で再現性があるか」を確認する
- **資金計画** — 初期投資・損益分岐点・回収期間を明確にする
- **FC展開の場合** — 加盟店管理・スーパーバイザー体制の設計が生命線
- **デジタルとの統合** — リアル出店とオンライン集客を最初から連動させる

## フランチャイズ展開の特徴

```
メリット
├── 加盟店の資本・人材を活用して素早く展開できる
├── ロイヤリティ収入が安定収益になる
└── ブランド認知が広がる

注意点
├── ブランドの品質管理が難しくなる
├── 加盟店との関係管理コストがかかる
└── 標準化されたオペレーションが前提
```

## こんな課題に効く

- 特定エリアに限定されており、地域展開・全国展開したい
- 自社単独の展開には資金・人材が足りない
- 既存の成功モデルを横展開したい
- リアルな顧客接点を作ることで信頼・ブランドを高めたい
