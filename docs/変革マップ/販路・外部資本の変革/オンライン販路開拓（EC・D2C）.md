## 概要

ECサイト・自社通販・SNS販売などのオンラインチャネルを構築・拡大し、
デジタル上での集客・販売・顧客関係を作る手段。
既存のリアル販路に依存せず、直接顧客にリーチできるのが特徴。

## アプローチの種類

| アプローチ | 内容 | 向いているケース |
|-----------|------|---------------|
| モール型EC | Amazon・楽天・Yahoo!ショッピングへの出店 | 即座に集客基盤を借りたい |
| 自社EC（D2C） | 自社サイトで直販（Shopify等） | ブランド構築・顧客データ取得を重視 |
| SNS販売 | Instagram・TikTok・LINEでの販売導線 | 若年層・特定趣味層へのリーチ |
| BtoB EC | 企業向けの受発注オンライン化 | 営業コスト削減・既存顧客の利便性向上 |
| オムニチャネル | リアル店舗とオンラインの統合 | 既存店舗のデジタル化・相互送客 |

## D2C（Direct to Consumer）のポイント

中間業者を介さず、消費者に直接販売するモデル。

- **メリット**：顧客データを自社で保有できる、利益率が高い、ブランドコントロールができる
- **課題**：集客を自分でやる必要がある（広告費・SEO・SNS運用が必要）
- **成功の鍵**：LTV（顧客生涯価値）の最大化設計＋リピート施策

## オンライン集客の主な手段

```
集客チャネル
├── 自然検索（SEO）— コンテンツ資産が積み上がる
├── 広告（リスティング・SNS広告）— 即効性あり、コストかかる
├── SNS（Instagram・TikTok等）— ファン化・認知拡大
├── メール・LINE— 既存顧客へのリピート促進
└── インフルエンサー・PR — 認知獲得・信頼形成
```

## こんな課題に効く

- リアル店舗・既存チャネルだけでは成長に限界がある
- 特定の地域・顧客層に閉じた販路から脱したい
- 顧客データを直接取得・活用できる体制を作りたい
- オンラインでの認知・集客の仕組みがない
