## 概要

社外の組織・個人と連携することで変革を加速する手段。
内部リソースだけでは動かしにくい変革を、外部の力で突破するアプローチ。

## パートナーの種類

| 種類 | 特徴 |
|------|------|
| コンサル | 戦略・設計の専門知識を借りる |
| ベンチャー協業 | 新技術・新サービスを取り込む |
| アライアンス | 同業・異業種と連携して価値を拡張する |
